    Jewellery box - Additional information

    Activity 1:

    Using Computer Aided Design (CAD)

    Description of activity:

    Students will learn to visualise the production process of a jewellery box and create a Gant chart that reflects this process.

    Instructions:

  • Using Fusion 360 design a jewellery box with the following dimensions: 140mm x 140mm x 90mm.
  • The box needs to have a simple lid you can open and close.
  • Everything else about the box, such as the design, thickness of material used etc, can be designed however you like.​
  • If you are stuck, or need help visualising the box, take a look at the resource you have been provided.
  • Submit the completed file in the appropriate google classroom tile.

    • Task analysis:

    This learning activity is designed to give students an opportunity to familiarise themselves with using Computer Aided Design (CAD). This activity address outcome P6.2 (evaluates and uses computer-based technologies in designing and producing) within the preliminary Design and Technology curriculum (NESA, 2013a). It supports the numeracy general capabilities of using measurements and spatial reasoning as students are required to use dimensions (measurements) in their CAD designs whilst understanding and visualising their product as a three dimensional object when designing (Australian curriculum, n.d.b). By establishing a challenging learning goal and implementing the use of ICT (AITSL, n.d.), students are able to apply prior knowledge on the subject in a new medium whilst learning a new skill which will help them develop competency in using CAD. Although this task doesn't exactly support higher-order thinking (Armstrong, 2010), it is a very beneficial activity for students at this stage which will provide an opportunity for tasks that support higher-order thinking later down the line.
